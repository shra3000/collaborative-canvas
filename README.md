\# Collaborative Canvas



A real-time collaborative drawing board built using Node.js, Socket.IO, and HTML5 Canvas.  

Multiple users can draw together, change color and brush size, and undo their latest strokes.



---



\## Features



\- Real-time collaborative drawing with Socket.IO  

\- Individual undo for each user  

\- Adjustable brush color and size  

\- Efficient canvas rendering for smooth performance  

\- Room-based collaboration architecture  



---



\## Tech Stack



\- Frontend: HTML, CSS, JavaScript (Canvas API)  

\- Backend: Node.js, Express, Socket.IO  

\- Version Control: Git and GitHub
